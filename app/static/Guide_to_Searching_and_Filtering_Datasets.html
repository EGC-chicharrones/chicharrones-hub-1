Guide to Searching and Filtering Datasets

Introduction

The dataset search section allows users to find and filter datasets using various options such as authors, affiliations, descriptions, ratings, and more. The interface is interactive, enabling users to specify their search criteria using simple keywords.

If no keyword is specified, the search will default to matching the title of the datasets. For example, typing Sample dataset 2 will search for datasets with "Sample dataset 2" in the title.

Keywords for Filtering Datasets

  1. Search by Author
  • Keyword: author
  • How it works: Finds datasets written by a specific author.
  • Example: Type author: Author 1 to search for datasets associated with Author 1.
  1. Search by Affiliation
  • Keyword: affiliation
  • How it works: Filters datasets based on the affiliation of the authors.
  • Example: Type affiliation: University of Example to find datasets whose authors are affiliated with this university.
  1. Search by ORCID
  • Keyword: orcid
  • How it works: Searches for datasets based on the author’s ORCID identifier.
  • Example: Type orcid: 0000-0001-2345-6789.
  1. Filter by DOI
  • Keyword: doi
  • How it works: Finds datasets using the DOI of the associated publication.
  • Example: Type doi: 10.1234/example to search for a specific dataset.
  1. Filter by Rating
  • Keyword: rating
  • How it works: Returns datasets with a minimum specified rating.
  • Example: Type rating: 4.0 to search for datasets rated 4 or higher.
  1. Filter by Tags
  • Keyword: tags
  • How it works: Finds datasets tagged with specific terms.
  • Example: Type tags: AI, Machine Learning to search for datasets related to AI and machine learning.
  1. Filter by Description
  • Keyword: description
  • How it works: Searches datasets whose descriptions contain the specified keywords.
  • Example: Type description: climate to find datasets related to climate.
  1. Filter by Anonymization Status
  • Keyword: anonymized
  • How it works: Filters datasets based on their anonymization status.
  • Example: Type anonymized: true to find anonymized datasets.
  1. Filter by Date
  • Keyword: date
  • How it works: Filters datasets created after a specific date.
  • Format: yyyy/mm/dd.
  • Example: Type date: 2023/01/01 to find datasets created after January 1, 2023.
  1. Sort by Date
  • Options: newest (most recent) or oldest (oldest first).
  • How it works: Organizes results by their creation date.
  • Example: Select newest to see the most recent datasets first.

Use Case Examples

  1. Search for datasets by a specific author
  • Type: author: Author 1.
  1. Filter by minimum rating and tags
  • Type: rating: 4.5; tags: physics, quantum.
  1. Combine affiliation and anonymization filters
  • Type: affiliation: Tech Institute; anonymized: false.
  1. Search for datasets created recently with a specific description
  • Type: description: neural networks; date: 2024/01/01.
  1. Sort results by the oldest first
  • Select oldest from the sorting options.

Additional Notes

  • If no keyword is provided, the search will match the dataset's title by default.
  • Separate keywords with a semicolon (;) when combining multiple filters.
  • Use these filters to refine your searches and quickly find the datasets you need.
